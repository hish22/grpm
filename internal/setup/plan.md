** Setup process **

Note: exe and other simailer files can't be operated in setup phase

# Section 1 (Setup compressed/archive file):
- after the download process completed ask user if he wants to setup the asset
-  if user agrees:
-   start unzip the file
-   add the extracted file/files into a dir if they are not
-   add the extracted file location to $PATH to refer from any place (linux,windows)
-  endif
# Section 2 (Setup dkpg files):
- after the download process completed ask user if he wants to setup the asset
-  if user agrees:
-   start by running dpkg command
-   let the dkpg process handle the installation setup
-  endif
